# phantom-clj

Playing around with ClojureScript + PhantomJS.

## Usage

Not encouraged.

You'll need PhantomJS installed. You can put it in your path, or use `-p /path/to/phantomjs`.

```
lein run example.cljs
```

See the included `example.cljs` for basic usage.
